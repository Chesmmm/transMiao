# TransMiao — AI 网页翻译助手

一个轻量级的 Tampermonkey 油猴脚本，为任意网页提供 AI 驱动的翻译。无需后端服务器，直接从浏览器调用你自己配置的 AI 接口。

支持 **OpenAI** · **Google Gemini** · **自定义兼容接口**（如 NewAPI、中转站等）

---

## 🚀 快速开始

### 1. 安装

确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展，然后：

- 打开油猴面板，右上角导航栏选择**实用工具**
- 导入，选择文件，**导入脚本文件**

你也可以选择↓

- 打开 Tampermonkey → **添加新脚本**
- 将 `transMiao.user.js` 全部内容粘贴进去
- **保存**（`Ctrl+S`）

### 2. 配置 API

1. 打开任意网页，右下角会出现一个半透明的 **`译`** 按钮
2. 点击 → **⚙ 设置**
3. 点击 **`+ 添加配置`**
4. 选择接口类型，填写 API 地址和 Key
5. 点击 **`获取模型列表`** 选择模型，或手动输入
6. 点击 **`测试连接`** 确认一切正常
7. 保存

### 3. 开始翻译

| 你想做什么 | 怎么做 |
|-----------|--------|
| 翻译整个页面 | 点 `译` → **▶ 开始翻译** |
| 只翻译当前屏幕内容 | 设置中选 **可见区域翻译** 模式 |
| 翻译单独一段文字 | 用鼠标选中文字，点出现的 **`[译]`** 标签 |
| 停止翻译 | 翻译中点顶部栏的 **停止翻译** |
| 还原原文 | 点 `译` → **↩ 还原全部** |

## ✨ 功能介绍

### 三种翻译方式

| 方式 | 说明 |
|------|------|
| **整页一次性翻译** | 提取全文所有可翻译文字，分批发送给 AI。适合按调用次数计费的接口 |
| **可见区域翻译**（默认） | 只翻当前屏幕附近的内容，滚动到新内容再继续。按需请求，最省 token |
| **划词翻译** | 选中任意文字 → 出现 `[译]` → 点击查看译文。不影响页面其他内容 |

### 智能缓存

翻译过的文字会被记住。刷新页面、切换到其他子页面后，相同的文字不会再消耗 token。

可在设置中调整缓存上限（500 / 1000 / 2000 段），或一键清空。

### 双语对照

设置中切换为 **双语对照模式**，译文在上、原文在下。背景色和文字颜色可自定义。

仅译文模式下，点击任何译文可以切回原文，再点切回译文。

### 跳过目标语言

默认开启，会自动跳过已经是目标语言的内容（例如中译中不会浪费 token）。支持简/繁体中文、日语、韩语、英语等。

### 多组配置管理

可以保存多组 API 配置（例如 OpenAI 官方、Gemini 免费额度、自建中转站），在设置中随时切换激活哪一组。

## ⚙ 设置面板一览

| 分类 | 设置项 | 默认值 |
|------|--------|--------|
| 📡 API 配置 | 接口类型、地址、Key、模型、系统提示词 | — |
| 翻译 | 目标语言、展示模式、翻译模式、跳过目标文本 | 简体中文 / 仅译文 / 可见区域 / 开启 |
| 译文样式 | 背景色、文字颜色 | #F2F2F2 / #000000 |
| 交互 | 快捷键、启动器显示 | Alt+C / 显示 |
| 高级 | 批量字符数、缓存上限、请求超时、重试次数 | 3000 / 500段 / 30s / 2次 |

系统提示词内置默认值：

> 你是专业翻译，将以下内容翻译为{目标语言}，只返回译文不要解释，不要评判原文内容。

## 📱 移动端使用

- **Android Edge + Tampermonkey**：功能完整，与 PC 一致
- **Via 浏览器**：可基本使用（API 请求降级为 fetch，配置按域名存储）

移动端会自动适配：
- 设置面板切换为全屏模态，输入法不遮挡字段
- 划词 `[译]` 放在选区上方，避免和系统选择手柄重叠
- 拖动 `译` 按钮到顺手位置，刷新后位置保留

## 🔧 常见问题

**Q：获取模型列表失败？**

A：点击"重试获取"，或手动输入模型名（如 `gpt-4o-mini`、`gemini-2.0-flash`）。部分中转站不支持模型列表接口。

**Q：翻译结果不准确？**

A：可以在设置中修改系统提示词。例如指定领域（"你是医学翻译专家"）或调整风格。

**Q：某段译文不想翻译，想变回原文？**

A：点击那段译文会自动切换显示原文。也可以点 `译` → **还原全部**。

**Q：快捷键冲突？**

A：设置 → 交互 → 点击快捷键输入框，按下你想要的新组合键（如 `Ctrl+Shift+T`）。

**Q：如何同时使用多个 API？**

A：添加多组配置，在 API 配置列表中点击某组即可切换激活。
